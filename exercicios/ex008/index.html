Principais formatações

Negrito/Destaque

Texto em negrito usando a tag < b > (Texto não semantico)

Texto em negrito usando a tag <strong> (Texto semantico)

Texto em Italico/ Ênfase

Texto em italico usando a tag <i> (Texto não semantico)

Texto em italico usando a tag <em > (Texto semantico )

Texto marcado

Texto marcado com a tag < mark > (A tag MARK não é reconhecida no VSCODE )
Podemos mudar a cor da marcação usando CSS: <mark style="background-color: "">.
Exemplo: marcação em azul

Texto grande e pequeno

Criando um texto grande e um texto pequeno.
Aumento da letra utilizando a tag < big > ( A tag big se tornará obsoleta a partir de algum dia ai )
Diminuir a fonte usando a tag <small>

Texto deletado

Marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado. Com a tag <del>

Texto inserido

Pode se marcar um texto como inserido para dar ênfase que foi adicionado posteriormente com a tag <ins>

Texto sublinhado

Podemos subinhar um texto com a tag <u> ( não semantica )

Texto sobrescrito

para escrever algo como porcentagem, Exemplo: x20+3. Com a tag <sup>.

Texto subscrito

Para inserir coisas como formulas quimicas H2O usando a tag <sub>.
