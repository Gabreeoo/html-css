Usando a tag <video>, podemos colocar videos no site. Também tendo metodos assim como a tag <media> mas podendo usar o width para alterar o tamanho

Também podemos usar o <source>, assim como nas fotos. Por exemplo:

<source src="midias/RONALDINHO x MANO ZAFE MANO ZAFE ULTRA BASS.mp4" type="video/mp4">
