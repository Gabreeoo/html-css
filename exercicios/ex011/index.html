Imagens dinamicas

Tente abrir o site de outros dispositivos, ou apenas aumente o tamanho do navegador.

A tag <source> usada para dar dinamica ao tamanho das imagens, tendo que seguir a ordem de tamanho.

Reprodução de Audio.

Utilizando a tag <audio>, podemos inserir a musica com a ajuda de alguns metodos, exemplo:
<audio src="midia/Jay-z - 444.mp3" controls autoplay>


Passaremos agora para outro metodo de colocar audios no site, com a mesma tag, pórem de forma diferente. 

            
                < audio preload="metadata" controls> 

                <source src"midia/Jay-Z - 444.mp3" type="audio/mpeg">
